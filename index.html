Shared language

展示和报告中怎样解释这些指标？

Use this as a quick metric explainer during Q&A: proxy metrics are useful visual signals, but paper-level claims still need evidence cards or award-citation checks.

Recognition lag

发表年份到获奖年份的间隔，说明“正式回看确认”花了多久。

Safe wording Measures delayed formal recognition, not when impact first appeared.

Citation depth

公开学术元数据中的引用规模，用来观察后续研究吸收强度。

Safe wording A useful influence signal, not the only definition of importance.

Impact breadth

基于 OpenAlex citing works 抽样估计跨领域、机构、国家和年份的扩散范围。

Safe wording An OpenAlex sampled proxy for diffusion, not a complete citation graph.

Same-field benchmark

把选中论文和同领域/同主题邻近论文比较，减少跨领域引用习惯误读。

Safe wording A contextual comparison within this dataset, not a universal ranking.

Topic label

由 API concepts 和规则归一化得到的初步主题标签，代表论文需要人工核查。

Safe wording Good for trend visualization; final case claims should use manual topic checks.

Evidence card

把代表论文的贡献、证据链接、展示状态和人工检查要求集中成一张卡。

Safe wording Use presentation-ready cards first; keep human-check cards cautious.
