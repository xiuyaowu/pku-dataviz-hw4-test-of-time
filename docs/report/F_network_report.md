## F Network 研究生态与全球扩散

### F.1 研究生态概览

长期影响论文所构成的研究生态呈现出明显的机构集聚特征，但并未形成由单一机构主导的垄断格局。根据当前数据集，共识别出169个机构节点，共同构成了长期影响研究的知识生产网络。其中，UC Berkeley（17篇）、CMU（15篇）、Stanford（14篇）以及MIT（11篇）等国际顶尖学术机构处于网络核心位置，在长期影响成果的产生与传播过程中发挥着重要作用。

从国家层面来看，共有23个国家和地区出现在样本中，形成了以美国为核心、多个国家共同参与的全球研究生态。整体而言，长期影响研究既表现出明显的地理集聚效应，也呈现出一定程度的国际扩散特征。

### F.2 国家分布特征：美国主导下的多中心格局

国家分布结果显示，美国在长期影响研究中占据显著优势地位。基于作者机构信息统计，美国相关记录达到170条，占全部国家记录的66.4%，反映出其在计算机科学及相关领域长期积累的科研实力与影响力。

#### 2.2 互动功能与价值：

**“Exclude USA"**  ：当排除美国样本后，全球研究生态的多样性特征变得更加清晰。加拿大成为最具代表性的非美国研究中心，共有24篇长期影响论文分布于11家机构，其中多伦多大学和英属哥伦比亚大学（UBC）表现尤为突出。英国紧随其后，共有17篇论文分布于10家机构，剑桥大学和牛津大学等传统研究型大学持续保持较高影响力。德国则以11篇论文构成欧洲大陆的重要研究节点。

此外，以色列虽然整体规模较小，但6篇论文分布于4家机构，体现出较高的科研密度和较强的单位产出能力。亚太地区的新加坡、韩国、中国和印度等国家虽然在数量上尚未形成显著规模，但已经展现出持续发展的研究网络结构。

**国家气泡图点击弹出顶级机构**：点击国家气泡弹出顶级机构信息，实现宏观国家分布与微观机构构成的无缝连接，支持用户从整体到局部的深度数据探索，更快速的了解到各个国家的顶级机构分布

总体而言，研究结果表明，尽管美国在长期影响研究领域保持主导地位，但全球研究生态并非单中心结构。多个国家和地区已经形成具有自身特色的研究力量，共同推动知识生产与传播。然而需要指出的是，本研究中的国家统计基于作者机构元数据，其结果反映的是机构归属的可见结构，而非国家间实际合作关系或贡献比例。

### F.3 机构类型分化：学术机构与工业实验室的双轨结构

进一步将研究机构划分为学术机构（Academic Institutions）和工业实验室（Industry Labs）后，可以观察到两类主体在长期影响研究中的不同角色定位。

在学术机构方面，UC Berkeley、CMU、Stanford 和 MIT构成最具影响力的第一梯队。这些机构不仅在论文数量上保持领先，也在知识扩散网络中占据核心位置。同时，加拿大的多伦多大学和UBC等机构也进入高影响力机构行列，显示出长期影响研究已超越美国本土范围，在国际学术体系中形成更广泛的扩散趋势。整体来看，学术研究力量虽然存在核心集聚现象，但其地理分布相对均衡，非美国机构亦具备较强影响力。

相比之下，工业实验室的分布则表现出更高程度的集中化特征。IBM Almaden Research Center与IBM总部均贡献8篇长期影响论文，位居工业机构前列.Intel和Nokia分别贡献7篇论文，构成第二梯队。从整体结构来看，美国科技企业在工业研究领域占据绝对主导地位，包括Microsoft、HP和AT&T等大型企业均具有较高可见度，而非美国工业实验室则相对较少出现。

这一差异表明，学术研究与工业研究在知识生产机制上存在明显区别。学术机构更强调基础研究和知识传播，因此更容易形成跨国扩散网络. 工业实验室则更多围绕技术创新与商业应用展开，其研究资源和影响力相对集中于少数大型企业。

但是，学术和工业部分还可以从另外一个角度出发：工业研究机构与顶尖大学共同出现在高频机构名单中，也可以发现计算机科学领域长期影响的产学研混合生态特征。这一模式揭示了计算机科学领域创新的独特路径：基础研究突破往往源于学术机构，而工业实验室在成果转化和规模化应用中扮演关键角色。大学与企业研究机构的紧密互动，形成了"基础探索→技术验证→产业落地"的完整链条。系统、数据库、网络等方向常见大学与工业实验室共同推动的现象，体现了产学研深度融合的特点。

### F.4 全球知识网络：从集中到扩散的演化路径

全球研究网络的结构特征进一步揭示了长期影响研究所遵循的“集中—扩散”模式。重大研究突破往往首先产生于少数具有强大科研能力的核心机构，例如Berkeley、CMU、Stanford和MIT等。随后，这些成果通过多种机制向全球范围传播并产生持续影响。

首先，学术引用构成知识扩散的基础路径。高影响力论文被不同国家和机构的研究者持续引用、验证和扩展，从而不断扩大其影响范围。其次，研究人员在机构之间的流动促进了知识与研究范式的跨区域传播。再次，国际合作项目为知识共享提供了组织化渠道，进一步加强了全球研究网络的连接性。

在当前数据集中，共识别出256条论文—国家连接关系，形成一个相互关联的知识传播网络。美国作为最重要的研究中心，通过这些连接与全球23个国家和地区形成关联，其影响力由此扩散至更广泛的国际研究社区。值得关注的是，英属维尔京群岛（VG）在平均引用表现上呈现出较强信号。这一结果提示，部分规模较小的地区或机构也可能在特定研究方向上产生极高影响力，从而突破传统规模优势所形成的认知框架。

### F.5 主要发现

研究结果表明，长期影响研究呈现出地理集中与国际扩散并存的特征。美国在整体影响力结构中占据核心地位，但多个国家和地区已经形成具有持续产出的研究力量。与此同时，学术机构与工业实验室表现出明显不同的发展路径：前者呈现国际化扩散趋势，后者则主要集中于美国大型科技企业。

从知识传播机制来看，长期影响成果通常遵循“核心机构产生突破—全球网络扩散影响”的发展模式，其传播过程依赖于学术引用、人员流动以及跨国合作等多重渠道。整体上，长期影响研究并非由单一国家或机构独占，而是在全球范围内形成了一个由核心节点驱动、多中心参与的知识生态系统。

### F.6 结果解释与数据边界

本部分分析基于OpenAlex提供的机构与国家元数据，因此相关结论应被理解为数据集中可观察到的结构性特征，而非对全球科研合作网络的完整刻画。机构和国家字段可能受到数据缺失、机构归并、署名方式差异等因素影响，从而产生一定偏差。

此外，国家统计基于作者所属机构信息，不代表所有作者的实际贡献比例，也不构成国家科研实力排名。工业实验室的出现反映的是其在元数据中的可见性，而非其技术成果的实际产业影响程度。对于缺失或不完整的元数据，应将其视为数据覆盖范围的限制，而不能据此推断不存在相关合作关系或研究活动。
